Parametrized Curve Renderer

https://github.com/phil-shenk/curve-generator-opengl

This project demonstrates the representation of an arbitrary parametrized 3D curve. The curve is defined by functions of x, y and z, and a 3D tube representation of the curve is generated and rendered in OpenGL. The camera can be rotated and zoomed with the mouse. Take a look at the repository's readme to see some of the math that goes into making the tube vertices consistent.

All of the linear transformations in this project make use of my linear algebra library, which can be found here: https://github.com/phil-shenk/linalglib

Rendering billiards

https://github.com/phil-shenk/billiards-opengl

This project demonstrates the generation and texturing of spherical meshes, a simple version of the Blinn-Phong lighting model, and fake shadows. The light source and the camera can be moved by the user's keyboard, and the camera can enter follow-mode to follow behind one of the billiard balls.

All of the linear transformations in this project make use of my linear algebra library, which can be found here: https://github.com/phil-shenk/linalglib

Parallel Erosion Simulation

https://github.com/phil-shenk/erosion

This project simulations hydraulic erosion with thousands of raindrops. Each drop absorbs or deposits a variable amount of sediment based on its speed and the slope of the terrain. The simulation makes use of python's multiprocessing library to leverage the extra processing power of multicore machines.
The simulation is rendered in a live, interactive window using the Mayavi package for python.

Maze demonstration

https://github.com/phil-shenk/maze-solver-opengl

This project demonstrates the generation and solving of a simple maze. After it is generated, the maze is rendered in 3D with OpenGL. The camera then moves through the maze in the order that it is traversed by the simple recursive solving algorithm. After the camera exits the far side, it walks back through the maze along the optimal path.

All of the linear transformations in this project make use of my linear algebra library, which can be found here: https://github.com/phil-shenk/linalglib
